Projects

Designed and implemented the high-level autonomous software architecture for a 2,000-lb, 12-ft-long tunnel boring machine capable of executing a 100-ft excavation. The system coordinated multiple subsystems to safely transition between manual, semi-autonomous, and fully autonomous operation.

I focused on decision-making logic, system state management, and trajectory execution, ensuring robust behavior under both nominal and fault conditions. The software was designed to be modular, testable, and responsive to real-time feedback.

  • Developed a finite state machine (FSM) to manage operational modes including idle, manual, safe manual, autonomous, and abort states
  • Implemented autonomous motion along a curved excavation trajectory using a guidance and trajectory-planning framework
  • Integrated safety logic to handle off-nominal conditions and enforce safe state transitions during faults
  • Coordinated software behavior across multiple subsystems including actuation, sensing, and supervisory control
  • Iterated the design through formal reviews and live testing, incorporating feedback to improve robustness, clarity, and fault tolerance

Designed and developed an interactive Plotly Dash application to visualize a planned excavation trajectory alongside real-time position updates. The application loads a predefined dig path from a CSV file and dynamically overlays live motion data as the system progresses.

The visualization was built to clearly distinguish between the intended trajectory and the actual motion of the system, enabling intuitive monitoring and debugging of autonomous behavior. The interface updates continuously while preserving performance and visual clarity.

  • Implemented 3D trajectory visualization comparing planned paths against live position data in real time
  • Added a dynamically updating path trace with a marker indicating the current system position
  • Integrated live updates from control system data using a timed refresh loop
  • Designed a clean, readable UI to support quick interpretation during testing and demonstrations
  • Structured the application for modularity, enabling easy extension to additional telemetry or control inputs

Choreographed an African dance performance over the course of an academic semester, culminating in a live stage production at the University of Colorado Boulder. The work blended traditional African movement vocabulary with contemporary elements to create a piece that emphasized rhythm, community, and cultural expression.

The process involved developing original choreography, rehearsing with a group of dancers, and refining the performance through iterative practice and feedback. Special attention was given to musical interpretation, group dynamics, and conveying belonging through movement.

  • Designed original choreography inspired by traditional West African dance styles and modern movement techniques
  • Led rehearsals over multiple weeks, coordinating timing, formations, and transitions
  • Adapted choreography to suit the strengths and experience levels of the dancers
  • Collaborated with performers to refine expression, musicality, and stage presence
  • Presented the final piece in a live performance setting, emphasizing cultural respect and authentic representation of African dance traditions.
